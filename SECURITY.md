# Security Policy

## Reporting
If you find a security issue, do not open a public issue first.

Use private disclosure to the repository owner and include:
1. Affected file/module
2. Reproduction steps
3. Impact assessment
4. Suggested fix (if available)

## Supported scope
This repository targets browser-based 2D games and build tooling.

Areas of interest:
1. Manifest/file path handling
2. Asset loading paths
3. Build and script execution behavior
4. Multiplayer code (when added in future versions)
